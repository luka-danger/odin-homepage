My Work

Job Raptor

Job Raptor is a full-stack app, built using React, Flask, MongoDB, and a Decision Tree ML model to classify job postings as real or fake. The Flask API serves predictions using a pre-trained model, and job postings are fetched from MongoDB via the API and displayed randomly on the React UI.

Alpine Running Depot

Alpine Running Depot is a full-stack Spring Boot application featuring an HTML front-end and Java back-end, utilizing an H2 SQL database to manage a fictitious online trail running supply store where users can add, delete, and update inventory, as well as access pages for signing up, logging in, and contacting support.

Schwifty Archives

Schwifty Archives is a Rick and Morty character explorer built with vanilla JS, CSS, and powered by the Rick and Morty API. It features pagination, search, modular architecture, and a smooth CSS ticker animation for an interactive experience.

To Do List

Hotel Scheduling App

The Landon Hotel Booking app is a modified full-stack solution for the Landon Hotel, featuring an Angular front-end built with TypeScript and a Java/Spring Boot back-end, enhanced with localization support and multithreading, and fully containerized and deployed using Docker.

Package Delivery Algorithm

Package Delivery Algorithm based on the Traveling Salesman Problem. The program finds the shortest distance to deliver all packages (117.3 miles) to various addresses in the Salt Lake City metro area.
